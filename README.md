# React ToDo

### Description

Simple todo app using react

---

### Links

- [Github](https://github.com/Kmg11/React_ToDo "Github Repo")

---

### Features

- Add task
- Delete task
- Complete task
- Complete all tasks
- No tasks message
- Cool Design

---

### Technologies

- React
- Css

---

### How To Run Website On Your Maachine

- You Need To Setup [ nodejs, gulp ] on your machine
- Clone Repo
- On Terminal

```bash
  npm install
```

```bash
  npm run start
```
